# 👟 Raang Shoes Zone

A beginner-friendly **shoe store website** built using **HTML, CSS, and JavaScript**.

## 🔥 Features

* 🛒 Add to cart interaction
* 💳 Purchase button with redirect page
* 🎨 Custom UI with banner and product layout
* 📱 Simple responsive design

## [🚀 Demo](https://gauraangcodes.github.io/shoes-website-practice/)



## 🛠️ Tech Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

* `index.html` → main product page
* `thankyouforpurchase.html` → confirmation page
* `style` → styling inside HTML

## 🎯 What I Learned

* How to structure a webpage using HTML
* Styling layouts using CSS (flexbox, spacing, alignment)
* Handling button clicks using JavaScript
* Page navigation using `window.location.href`
